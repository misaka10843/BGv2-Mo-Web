# BanG Player v2

一款《BanG Dream! 少女乐团派对》web版模拟器。

此仓库并不是网页端，需配合[BPv2-Mo-Adnroid](https://github.com/misaka10843/BPv2-Mo-Adnroid)

## 安装依赖
```bash
yarn
```

### 开发模式
```bash
quasar dev
```

### 文件分析
```bash
yarn run lint
```

### 构建发布应用
```bash
quasar build
```

### 自定义配置
查看[Configuring quasar.conf.js](https://quasar.dev/quasar-cli/quasar-conf-js).
